Имея список элементов, создайте и верните список, элементы которого являются списками, которые содержат последовательные серии одинаковых элементов исходного списка. Обратите внимание, что элементы, которые не дублируются в исходном списке, должны в результате стать одноэлементными списками, чтобы каждый элемент был включен в итоговый список списков.

Входные данные: {% if interpreter.slug == "js-node" %} Массив строк (strs) и целых чисел (ints). {% else %} Список строк (strs) и целых чисел (ints). {% endif %}

Выходные данные: {% if interpreter.slug == "js-node" %} Массив массивов строк (strs) и целых чисел (ints). {% else %} Список списков строк (strs) и целых чисел (ints). {% endif %}

Пример:

{% if interpreter.slug == "js-node" %} 
groupEqual([1, 1, 4, 4, 4, "hello", "hello", 4]) == [[1,1],[4,4,4],["hello","hello"],[4]]
groupEqual([1, 2, 3, 4]) == [[1], [2], [3], [4]]
groupEqual([1]) == [[1]]
groupEqual([]) == []
{% else %} 
group_equal([1, 1, 4, 4, 4, "hello", "hello", 4]) == [[1,1],[4,4,4],["hello","hello"],[4]]
group_equal([1, 2, 3, 4]) == [[1], [2], [3], [4]]
group_equal([1]) == [[1]]
group_equal([]) == []
{% endif %}
